1. Após o repositório ser inicializado com `git init`, foi criado um arquivo README.md por meio de `echo "# Projeto Totem Check-in" > README.md` e então commitado com `git commit -m "chore: initial commit"`, para preencher a main.
2. É criada a develop a partir da main com `git checkout -b develop main`. Então são criadas as estruturas de features com `git checkout -b feat/seat-selection`, de releases com `git checkout -b release/1.0.0 develop`, e de hotfixes com `git checkout -b hotfix/fix-printer-error main`.
3. Na branch `feat/seat-selection` começou a ser desenvolvida uma feature. Foram criados dois arquivos por meio de `echo "Logica de assentos" > seat-map.js` e `echo "Estilos do mapa" > seat-map.css`. Assim feito o commit por meio de `git commit -m "feat: implement seat map visualization and selection"`
4. A feature é integrada na develop por meio de um merge `git merge --no-ff feat/seat-selection -m "merge: integrate seat selection feature into develop"`. Assim a branch `feat/seat-selection` é deletada com `git branch -d feat/seat-selection`. E o remoto é atualizado com `git push origin develop`
5. Com o conteúdo de develop já na release/1.0.0, é feito o merge da release na main com `git merge --no-ff release/1.0.0 -m "chore: release 1.0.0"` e criada a tag com `git tag -a v1.0.1 -m "Versão oficial de lançamento do Totem de Check-in"` e é deletada a branch release/1.0.0 com `git branch -d release/1.0.0`.
6. Na branch `hotfix/fix-printer-error` é feita a "correção" de um do arquivo com `echo "function printTicket() { console.log('Correção: Timeout de impressão resolvido'); }" > printer-fix.js` e commitado com `git commit -m "fix: resolve critical timeout on ticket printing"`. Assim realiado o merge na main por `git merge --no-ff hotfix/fix-printer-error -m "fix: apply hotfix 1.0.1"` e criada a tag `git tag -a v1.0.2 -m "Versão de correção 1.0.2"` e sincronizado no develop `git merge --no-ff hotfix/fix-printer-error -m "merge: sync hotfix 1.0.2 to develop"` e finalizado `git branch -d hotfix/fix-printer-error`.
